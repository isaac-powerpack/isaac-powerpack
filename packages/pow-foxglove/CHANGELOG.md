# isaac-powerpack version history

## 0.1.0a3 (2026-01-26)

### Features

- add `pow: Teleop` panel to control robot 6-DOF movement velocity (Twist messages) using keyboard inputs
- rename isaac-powerpack package to pow-foxglove

### Bug Fixes

- fix Canvas component vertical resize issue

## 0.0.1 (2025-12-11)

### Features

- add Detection2D panel to display 2D bounding boxes from `vision_msgs/msg/Detection2DArray` messages
- display object IDs, confidence scores, and label information on the Detection2D panel
- support custom object labels and IDs through Foxglove variables
